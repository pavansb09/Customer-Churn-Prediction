# Customer Churn Prediction

## Project Overview

This project predicts whether a telecom customer is likely to churn using machine learning techniques. The dataset used is the IBM Telco Customer Churn dataset.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost
- LightGBM
- Joblib
- Power BI

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- LightGBM

## Best Model

- Logistic Regression
- Accuracy: 78.75%
- ROC-AUC: 0.832

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Importance Analysis
- Model Comparison
- Customer Churn Prediction
- Interactive Power BI Dashboard
